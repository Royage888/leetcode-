## 								Leetcode No.7 回文数

**解题思路一**：

输入整数时分为三种情况：负数，正数回文数，和负数回文数

所以分情况讨论：负数时`return false`,正数时将数字反转，与原来比较，相同返回`return true`,不同返回`return false`

```java
class Solution {
    public boolean isPalindrome(int x) {
        int pas = x;
        int res = 0;
        if (x < 0) {
            return false;
        } else {
            while (x != 0) {
                int t = x % 10;
                res = res * 10 + t;
                x = x / 10;
            }
            if (res == pas) {
                return true;
            } else {
               return false;
        }
    }
}
}
```
